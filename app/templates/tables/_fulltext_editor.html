{# tables/_fulltext_editor.html — rich-text editor for an act's full text. Shared by the standalone /tables/fulltext page AND the admin edit-hub "Πλήρες κείμενο" panel so both behave identically (the repo keeps the two full-text UIs in sync). Expects `act` (adam, full_text, full_text_html) in context. Posts to /tables/fulltext/save, which stores plain text (full_text, the search source) and sanitised HTML (full_text_html) together. Works both on a normal page load and when injected via an HTMX swap (the admin panel): the loader is async-safe and the init is idempotent, so re-loading the panel (tab switching) re-initialises cleanly. Uses CLASSES, not ids, so it is safe even if ever included twice on one page. #}
