{# CSRF token delivery. Include once in . The session token (set by AuthMiddleware, exposed via _auth_context) is echoed on every unsafe request three ways, so no per-form/per-call edits are needed: 1. HTMX — X-CSRF-Token header on every request (htmx:configRequest); 2. fetch — X-CSRF-Token header on same-origin non-GET requests; 3. plain
— a hidden csrf_token field injected at submit. Server enforcement lives in csrf_protect (app/main.py). #}
